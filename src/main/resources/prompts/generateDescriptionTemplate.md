# 任务
为提供的景点、餐厅或美食等生成简洁的描述信息。

# 输入格式
一个包含地点/美食名称的列表，例如：
[中南大学, 岳麓山]

# 输出要求
1. **描述长度**：每个描述控制在150-200字左右
2. **描述内容**：应包含核心特色、历史背景、知名度等关键信息
3. **输出格式**：返回标准的JSON数组格式
4. **保持对应**：输出的描述数量和顺序必须与输入完全一致

# 输出格式说明
- 使用标准JSON数组格式：["描述1", "描述2", "描述3"]
- 必须是合法的JSON格式，可以直接被程序解析
- 不要添加任何JSON之外的说明文字或markdown代码块标记
- 描述文本内如果有双引号，需要正确转义
- 整个JSON数组不换行，保持紧凑格式
